Journey in Hacktiv 8

Hai people, Kenalin gw Fadli. Gw masuk di Hacktiv8 sebagai Batch 34. Hacktiv8 adalah sebuah pelatihan coding fulltime di Jakarta yang membantu pemula untuk menjadi fullstack developer dalam 12 minggu. Gw bukan dari background IT tapi gw percaya kalo dengan mengikuti program di Hacktiv8, Gw bisa menjadi seorang Programmer. Awal mula phase 0, JUJUR, Gw struggling banget butI enjoy it. Minggu pertama di phase 0 gw diajarin cara membuat website sebuah HTML dan CSS dan Akhirnya Voilaa, Gw sekarang udah lumayan bisa membuat website ( tapi masih basic banget but im proud that i can make a website). Gw terkadang down melihat teman-teman yang lain yang super duper udah bisa pake banget so "I trust the proccess" mengikuti semua perintah dan alur pembelajaran di Hacktiv8 dan ya Wish me luck
